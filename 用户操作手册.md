## 系统简介

这是一个现代化的个人博客系统，采用粉色系可爱风格设计，支持 PC 端和移动端访问。系统提供文章浏览、评论互动、用户管理等功能。

**技术栈：** Vue 3 + TypeScript + Element Plus + Vite

**访问地址：** `http://your-domain/web/`

## 访客功能

### 1. 浏览文章

#### 1.1 首页浏览
- **访问路径：** 首页 `/`
- **功能说明：**
  - 查看最新发布的 3 篇文章
  - 显示文章封面、标题、分类、标签
  - 显示阅读数、点赞数、评论数
  - 点击文章卡片可进入文章详情

#### 1.2 文章列表
- **访问路径：** 导航栏 → 文章 `/articles`
- **功能说明：**
  - 查看所有文章列表
  - 支持关键词搜索文章标题
  - 分页浏览（每页 10 篇）
  - 显示文章封面、标题、分类、标签、阅读数、点赞数

**操作步骤：**
1. 点击顶部导航栏的「文章」
2. 在搜索框输入关键词，点击「搜索」按钮
3. 点击文章卡片进入详情页

#### 1.3 文章详情
- **访问路径：** 点击文章卡片 `/article/:id`
- **功能说明：**
  - 阅读完整文章内容
  - 查看文章标签、分类、发布时间
  - 查看阅读数、点赞数
  - 支持 Markdown 格式渲染
  - 代码高亮显示
  - 图片自动适配

**操作步骤：**
1. 在首页或文章列表点击文章卡片
2. 滚动页面阅读文章内容
3. 点击右侧悬浮按钮进行点赞、收藏、分享（需登录）

### 2. 分类浏览

- **访问路径：** 导航栏 → 分类 `/category`
- **功能说明：**
  - 查看所有文章分类
  - 显示每个分类的文章数量
  - 点击分类卡片查看该分类下的所有文章

**操作步骤：**
1. 点击顶部导航栏的「分类」
2. 浏览所有分类卡片
3. 点击感兴趣的分类进入文章列表

### 3. 标签浏览

- **访问路径：** 导航栏 → 标签 `/tag`
- **功能说明：**
  - 查看所有文章标签
  - 标签云展示（字体大小表示文章数量）
  - 点击标签查看相关文章

**操作步骤：**
1. 点击顶部导航栏的「标签」
2. 点击任意标签
3. 在标签文章列表页查看相关文章
4. 支持表格视图和卡片视图切换（移动端自动切换为卡片视图）

### 4. 说说动态

- **访问路径：** 导航栏 → 说说 `/talk`
- **功能说明：**
  - 查看博主发布的动态说说
  - 支持图片、文字内容
  - 显示发布时间
  - 支持点赞和评论

### 5. 留言板

- **访问路径：** 导航栏 → 留言板 `/guestbook`
- **功能说明：**
  - 查看所有访客留言
  - 发表留言（需登录）
  - 回复留言（需登录）
  - 支持表情、图片上传

### 6. 友情链接

- **访问路径：** 导航栏 → 友链 `/link`
- **功能说明：**
  - 查看友情链接列表
  - 显示网站名称、描述、头像
  - 点击链接访问友站

### 7. 关于页面

- **访问路径：** 导航栏 → 关于 `/about`
- **功能说明：**
  - 查看博主介绍
  - 了解网站信息
  - 联系方式

## 用户功能

### 1. 用户注册

- **访问路径：** 登录页 → 立即注册 `/register`
- **功能说明：**
  - 创建新用户账号
  - 设置用户名、昵称、密码

**操作步骤：**
1. 点击顶部导航栏的「登录」按钮
2. 在登录页点击「立即注册」
3. 填写注册信息：
   - **用户名**：3-20 个字符，用于登录
   - **昵称**：2-20 个字符，显示名称
   - **密码**：至少 6 位字符
   - **确认密码**：再次输入密码
4. 勾选「我已阅读并同意用户协议和隐私政策」
5. 点击「注册」按钮
6. 注册成功后自动跳转到登录页

### 2. 用户登录

- **访问路径：** 顶部导航栏 → 登录 `/login`
- **功能说明：**
  - 使用用户名和密码登录
  - 支持记住密码功能

**操作步骤：**
1. 点击顶部导航栏的「登录」按钮
2. 输入用户名和密码
3. （可选）勾选「记住密码」
4. 点击「登录」按钮
5. 登录成功后返回之前浏览的页面

### 3. 文章互动

#### 3.1 点赞文章
- **访问路径：** 文章详情页
- **功能说明：**
  - 为喜欢的文章点赞
  - 再次点击取消点赞
  - 实时更新点赞数

**操作步骤：**
1. 登录后进入文章详情页
2. 点击右侧悬浮的「点赞」按钮（💗）
3. 点赞成功后按钮变为红色
4. 再次点击可取消点赞

#### 3.2 收藏文章
- **访问路径：** 文章详情页
- **功能说明：**
  - 收藏喜欢的文章
  - 在「我的收藏」中查看

**操作步骤：**
1. 登录后进入文章详情页
2. 点击右侧悬浮的「收藏」按钮（⭐）
3. 收藏成功后显示提示

#### 3.3 分享文章
- **访问路径：** 文章详情页
- **功能说明：**
  - 复制文章链接到剪贴板
  - 方便分享给朋友

**操作步骤：**
1. 进入文章详情页
2. 点击右侧悬浮的「分享」按钮（🔗）
3. 链接自动复制到剪贴板
4. 粘贴链接分享给朋友

### 4. 评论功能

#### 4.1 发表评论
- **访问路径：** 文章详情页底部
- **功能说明：**
  - 对文章发表评论
  - 支持文字、表情、图片
  - 支持 Markdown 格式

**操作步骤：**
1. 登录后进入文章详情页
2. 滚动到页面底部评论区
3. 在输入框中输入评论内容
4. （可选）点击表情按钮插入表情
5. （可选）点击图片按钮上传图片（最多 3 张）
6. 点击「发表评论」按钮

#### 4.2 回复评论
- **访问路径：** 文章详情页评论区
- **功能说明：**
  - 回复其他用户的评论
  - 支持多级回复

**操作步骤：**
1. 登录后在评论区找到要回复的评论
2. 点击评论下方的「回复」按钮
3. 在弹出的输入框中输入回复内容
4. 点击「发表回复」按钮

#### 4.3 删除评论
- **访问路径：** 文章详情页评论区
- **功能说明：**
  - 删除自己发表的评论
  - 博主可以删除任何评论

**操作步骤：**
1. 找到自己发表的评论
2. 点击评论右上角的「删除」按钮
3. 确认删除操作

### 5. 个人中心

- **访问路径：** 顶部导航栏 → 头像下拉菜单 → 个人中心
- **功能说明：**
  - 查看个人信息
  - 修改头像
  - 修改资料
  - 修改密码

#### 5.1 修改头像
**操作步骤：**
1. 点击顶部导航栏的用户头像
2. 选择「个人中心」
3. 点击头像区域
4. 选择本地图片文件（支持 jpg、png、gif，最大 5MB）
5. 上传成功后头像自动更新

#### 5.2 修改资料
**操作步骤：**
1. 在个人中心弹窗中点击「修改资料」按钮
2. 在弹出的表单中修改：
   - **昵称**：显示名称
   - **手机号码**：联系方式（可选）
   - **邮箱**：邮箱地址（可选）
3. 点击「保存」按钮
4. 修改成功后自动更新显示

#### 5.3 修改密码
**操作步骤：**
1. 在个人中心弹窗中点击「修改密码」按钮
2. 在弹出的表单中填写：
   - **旧密码**：当前密码
   - **新密码**：至少 6 位字符
   - **确认密码**：再次输入新密码
3. 点击「确定」按钮
4. 修改成功后自动退出登录，需要重新登录

### 6. 我的收藏

- **访问路径：** 顶部导航栏 → 头像下拉菜单 → 我的收藏 `/favorites`
- **功能说明：**
  - 查看收藏的文章列表
  - 快速访问收藏的文章
  - 取消收藏

**操作步骤：**
1. 点击顶部导航栏的用户头像
2. 选择「我的收藏」
3. 浏览收藏的文章列表
4. 点击文章卡片进入详情页
5. 点击「取消收藏」按钮移除收藏

### 7. 退出登录

**操作步骤：**
1. 点击顶部导航栏的用户头像
2. 选择「退出登录」
3. 确认退出后返回首页

## 博主功能

### 1. 管理后台

- **访问路径：** 顶部导航栏 → 头像下拉菜单 → 管理后台
- **功能说明：**
  - 跳转到管理后台系统
  - 管理文章、分类、标签
  - 管理评论、用户
  - 网站配置

**操作步骤：**
1. 使用博主账号登录
2. 点击顶部导航栏的用户头像
3. 选择「管理后台」
4. 在新标签页中打开管理后台
5. 自动携带登录状态，无需重复登录

**管理后台地址：** `http://your-domain/admin`

### 2. 博主权限

博主账号拥有以下特殊权限：
- 访问管理后台
- 发布、编辑、删除文章
- 管理分类和标签
- 删除任何评论
- 管理用户
- 修改网站配置
- 查看文章时不增加阅读数

## 移动端使用

### 1. 移动端特性

系统完全支持移动端访问，提供了优化的移动端体验：

#### 1.1 导航菜单
- **左侧汉堡菜单（☰）**：点击打开侧边栏导航
- **右侧用户头像/登录按钮**：快速访问用户功能

**操作步骤：**
1. 点击左上角的汉堡菜单按钮（☰）
2. 侧边栏从左侧滑出
3. 点击导航项跳转到对应页面
4. 侧边栏自动关闭

#### 1.2 文章阅读
- 文章内容自动适配屏幕宽度
- 图片自动缩放
- 代码块支持横向滚动
- 字体大小优化为 15px

#### 1.3 文章操作
- 点赞、收藏、分享按钮固定在右下角
- 按钮大小优化为 50x50px
- 触摸目标符合移动端标准（≥44px）

#### 1.4 评论功能
- 输入框自动适配屏幕宽度
- 图片上传按钮 3 列布局
- 提交按钮宽度 100%
- 支持移动端键盘

#### 1.5 表单输入
- 登录/注册表单输入框高度 45px
- 按钮高度 45px
- 字体大小 14-15px
- 触摸友好

### 2. 移动端操作技巧

#### 2.1 快速返回顶部
- 在文章详情页，点击右下角的「返回顶部」按钮（如有）
- 或使用浏览器的返回顶部功能

#### 2.2 横屏浏览
- 支持横屏模式
- 布局自动适配
- 提供更好的阅读体验

#### 2.3 分享文章
- 点击分享按钮复制链接
- 使用系统分享功能分享给朋友

### 3. 移动端浏览器兼容

系统支持以下移动端浏览器：
- ✅ Safari（iOS）
- ✅ Chrome（Android）
- ✅ 微信内置浏览器
- ✅ QQ 浏览器
- ✅ UC 浏览器
- ✅ 其他主流浏览器

## 常见问题

### 1. 登录相关

**Q: 忘记密码怎么办？**
A: 目前系统暂不支持自助找回密码，请联系博主重置密码。

**Q: 登录后为什么还是显示未登录？**
A: 请检查浏览器是否禁用了 Cookie，或尝试清除浏览器缓存后重新登录。

**Q: 可以使用第三方账号登录吗？**
A: 目前系统仅支持用户名密码登录，暂不支持第三方登录。

### 2. 文章相关

**Q: 为什么看不到某些文章？**
A: 可能是文章被设置为私密或草稿状态，只有博主可以查看。

**Q: 文章中的图片加载很慢怎么办？**
A: 这可能是网络问题，建议切换到更稳定的网络环境，或稍后再试。

**Q: 可以下载文章吗？**
A: 目前系统不支持文章下载功能，您可以使用浏览器的打印功能保存为 PDF。

### 3. 评论相关

**Q: 为什么我的评论没有显示？**
A: 评论可能需要博主审核后才能显示，请耐心等待。

**Q: 可以编辑已发表的评论吗？**
A: 目前系统不支持编辑评论，如需修改请删除后重新发表。

**Q: 评论支持哪些格式？**
A: 评论支持纯文本、表情符号和图片，支持 Markdown 格式（如有启用）。

### 4. 个人中心相关

**Q: 修改头像后为什么没有立即更新？**
A: 可能是浏览器缓存问题，请尝试刷新页面（Ctrl+F5 或 Cmd+Shift+R）。

**Q: 可以注销账号吗？**
A: 目前系统不支持自助注销账号，如需注销请联系博主。

### 5. 移动端相关

**Q: 移动端菜单打不开怎么办？**
A: 请尝试刷新页面，或清除浏览器缓存后重试。

**Q: 移动端文字太小看不清？**
A: 可以使用浏览器的缩放功能放大页面，或在浏览器设置中调整字体大小。

**Q: 移动端上传图片失败？**
A: 请检查图片大小是否超过 5MB，格式是否为 jpg、png、gif。

### 6. 其他问题

**Q: 系统支持哪些浏览器？**
A: 系统支持所有现代浏览器，包括 Chrome、Firefox、Safari、Edge 等。建议使用最新版本浏览器以获得最佳体验。

**Q: 如何联系博主？**
A: 可以通过以下方式联系博主：
- 在留言板留言
- 查看「关于」页面的联系方式
- 通过页脚的联系信息

**Q: 系统有 APP 吗？**
A: 目前系统仅提供网页版，但移动端浏览器访问体验已经过优化，可以添加到主屏幕作为 Web App 使用。
