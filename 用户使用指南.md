# 博客系统使用指南

欢迎使用本博客系统！这是一份简单易懂的使用指南，帮助您快速上手。

---

## 🚀 快速开始

### 访问网站
在浏览器中输入网址即可访问博客首页。

### 浏览文章
无需登录，直接点击文章卡片即可阅读。

---

## 📱 基础操作

### 1. 导航菜单

**PC 端：**
- 顶部导航栏包含：首页、文章、分类、标签、说说、留言板、友链、关于
- 点击任意菜单项即可跳转

**移动端：**
- 点击左上角的 **☰** 按钮打开菜单
- 点击菜单项跳转后，菜单自动关闭

### 2. 阅读文章

#### 在首页阅读
1. 首页展示最新的 3 篇文章
2. 点击文章卡片进入详情页
3. 滚动页面阅读完整内容

#### 在文章列表阅读
1. 点击导航栏的「文章」
2. 浏览所有文章列表
3. 使用搜索框查找感兴趣的文章
4. 点击文章卡片进入详情页

#### 按分类浏览
1. 点击导航栏的「分类」
2. 选择感兴趣的分类
3. 查看该分类下的所有文章

#### 按标签浏览
1. 点击导航栏的「标签」
2. 点击任意标签
3. 查看带有该标签的所有文章

## 👤 用户注册与登录

### 注册新账号

1. 点击右上角的「登录」按钮
2. 在登录页面点击「立即注册」
3. 填写注册信息：
   - **用户名**：用于登录（3-20 个字符）
   - **昵称**：显示名称（2-20 个字符）
   - **密码**：至少 6 位字符
   - **确认密码**：再次输入密码
4. 勾选同意用户协议
5. 点击「注册」按钮
6. 注册成功后跳转到登录页

### 登录账号

1. 点击右上角的「登录」按钮
2. 输入用户名和密码
3. （可选）勾选「记住密码」
4. 点击「登录」按钮
5. 登录成功后返回之前的页面

---

## 💬 互动功能（需登录）

### 1. 点赞文章

1. 登录后进入文章详情页
2. 点击右侧的「点赞」按钮（💗）
3. 点赞成功后按钮变红
4. 再次点击可取消点赞

**移动端：** 点赞按钮在右下角

### 2. 收藏文章

1. 登录后进入文章详情页
2. 点击右侧的「收藏」按钮（⭐）
3. 收藏成功后显示提示
4. 在「我的收藏」中查看收藏的文章

**查看收藏：**
- 点击右上角头像
- 选择「我的收藏」

### 3. 分享文章

1. 进入文章详情页
2. 点击右侧的「分享」按钮（🔗）
3. 链接自动复制到剪贴板
4. 粘贴链接分享给朋友

### 4. 发表评论

1. 登录后进入文章详情页
2. 滚动到页面底部评论区
3. 在输入框中输入评论内容
4. （可选）点击表情按钮添加表情
5. （可选）点击图片按钮上传图片（最多 3 张，每张最大 5MB）
6. 点击「发表评论」按钮

### 5. 回复评论

1. 找到要回复的评论
2. 点击评论下方的「回复」按钮
3. 输入回复内容
4. 点击「发表回复」按钮

### 6. 删除评论

1. 找到自己发表的评论
2. 点击评论右上角的「删除」按钮
3. 确认删除

## 🎨 个人中心

### 打开个人中心

1. 点击右上角的头像
2. 选择「个人中心」
3. 弹出个人信息窗口

### 修改头像

1. 在个人中心点击头像
2. 选择本地图片文件
3. 支持格式：jpg、png、gif
4. 文件大小：最大 5MB
5. 上传成功后头像自动更新

### 修改个人资料

1. 在个人中心点击「修改资料」按钮
2. 修改以下信息：
   - **昵称**：显示名称
   - **手机号码**：联系方式（可选）
   - **邮箱**：邮箱地址（可选）
3. 点击「保存」按钮
4. 修改成功后自动更新

### 修改密码

1. 在个人中心点击「修改密码」按钮
2. 填写以下信息：
   - **旧密码**：当前密码
   - **新密码**：至少 6 位字符
   - **确认密码**：再次输入新密码
3. 点击「确定」按钮
4. 修改成功后需要重新登录

### 退出登录

1. 点击右上角的头像
2. 选择「退出登录」
3. 确认退出

## ❓ 常见问题

### 登录问题

**Q: 忘记密码怎么办？**  
A: 请联系博主重置密码。

**Q: 登录后还是显示未登录？**  
A: 请检查浏览器是否禁用了 Cookie，或清除缓存后重新登录。

### 评论问题

**Q: 评论没有显示？**  
A: 评论可能需要审核，请耐心等待。

**Q: 可以编辑评论吗？**  
A: 目前不支持编辑，如需修改请删除后重新发表。

**Q: 评论支持什么格式？**  
A: 支持纯文本、表情符号和图片。

### 图片问题

**Q: 上传图片失败？**  
A: 请检查：
- 图片大小是否超过 5MB
- 格式是否为 jpg、png、gif
- 网络连接是否正常

**Q: 图片加载很慢？**  
A: 可能是网络问题，建议切换到更稳定的网络。

### 移动端问题

**Q: 移动端菜单打不开？**  
A: 请尝试刷新页面，或清除浏览器缓存。

**Q: 文字太小看不清？**  
A: 可以使用浏览器的缩放功能放大页面。
